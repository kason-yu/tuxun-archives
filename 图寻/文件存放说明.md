## 目录说明

### 根目录
- `index.html` - 项目的入口页面，包含所有功能模块的导航链接

### `assets/` - 静态资源文件夹
存放从根目录移出的 HTML 页面和通用图片文件，包括：9个功能页面（车牌、国旗、电线杆、加油站、语言类等）和通用背景图。

### `world_img/` - 世界影像
存放影像查询功能相关的 HTML 页面和大量图片资源。

### `world_map/` - 世界地图
存放地图相关功能（国家筛选、国家相似度）的 HTML 页面。

### `world_number/` - 世界手机号
存放世界手机号查询功能的 HTML 页面。

### `backend/` - 后端服务
Node.js 后端服务代码，提供数据接口支持。
